Notes for Marker

  • Running the Application: Due to browser security policies (CORS) for loading 3D models and ES6 modules locally (file:/// protocol), the application must be run via a local web server.
    • Option A: Using Python's built-in server: Navigate to the luck-lab-project directory in the terminal and run python -m http.server (for Python 3) or python -m SimpleHTTPServer (for Python 2). Then access via http://localhost:8000.
    • Option B: Using VS Code with the "Live Server" extension.
